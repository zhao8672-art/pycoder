# PyCoder 项目记忆

## 项目概况
- PyCoder：基于 Aider 二次开发的开源 AI 编程 Agent
- 版本：0.3.0-beta，Apache 2.0 许可证
- 核心代码量：~19,625 行 Python
- 技术栈：Textual TUI + FastAPI Server + Electron IDE + Vue 3
- 支持：DeepSeek/Qwen/GLM/Ollama 四家国产模型

## 商业化方向（2026-06-29 评估）
- 主策略：开源社区 → 企业版（方案A）
- 辅策略：金融/政务私有部署（方案C）
- 基础设施：Provider 统一接口作为 API 网关（方案B）
- 核心差异化：Python 垂直深度 + 国产模型原生 + BYOK 安全 + 开源透明
- 不做：通用IDE、免费烧钱获客、闭源

## 竞品格局
- 中国市场：Trae 41.2%（免费）、通义灵码 18.5%、文心快码 12.3%
- 全球：Copilot 470万付费用户、Cursor ARR 20亿美元、Claude Code 满意度最高
- 70% 开发者同时使用 2-4 个工具（多工具叠加趋势）
