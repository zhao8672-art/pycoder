# PyCoder Desktop UI 重新设计规格

## 一、设计目标

> 简洁、方便、灵活、好用

### 核心原则
1. **减少认知负担** - 从11个扁平活动项整合为4-5个逻辑分组
2. **聚焦核心任务** - 编辑器优先，AI对话作为核心工作流的一部分
3. **灵活布局** - 面板可调整大小、可拖拽
4. **快捷访问** - 低频功能通过命令面板访问

---

## 二、信息架构重构

### 当前问题
| 问题 | 影响 |
|------|------|
| ActivityBar 有11个图标，过于拥挤 | 用户难以快速定位功能 |
| 每个功能独立侧边栏面板 | 切换成本高，相关功能分散 |
| AI面板固定380px宽度 | 浪费空间或空间不足 |
| 所有功能都在侧边栏 | 低频功能占据宝贵空间 |

### 新架构：功能分组

```
┌─────────────────────────────────────────────────────────┐
│                    ActivityBar (简化为5组)               │
├─────────────────────────────────────────────────────────┤
│  📁 代码     🤖 AI      ▶ 运行     🌐 协作     ⚙ 命令   │
│  Files      Chat      Run       Team      Cmd+K       │
└─────────────────────────────────────────────────────────┘
```

#### 分组详情

| 分组 | 图标 | 包含功能 | 显示方式 |
|------|------|----------|----------|
| **代码** | 📁 | 文件树、搜索 | 侧边栏 |
| **AI** | 🤖 | AI对话、Hermes模式 | 右侧面板 |
| **运行** | ▶ | Python运行器、Run&Fix、测试生成、调试 | 底部面板 |
| **协作** | 🌐 | Git、团队协作、云端 | 侧边栏 |
| **命令** | ⚙ | 设置、技能市场、MCP工具 | Cmd+K面板 |

## 四、组件重构计划

### 4.1 ActivityBar 重构

**当前**: 11个扁平图标，使用emoji

**新设计**: 
- 5个逻辑分组图标
- 使用SVG图标替代emoji
- 悬停显示工具提示
- 激活状态显示下划线

### 4.2 Sidebar 重构

**当前**: 每个活动项对应一个独立面板

**新设计**:
- 按分组显示标签页
- 代码组: 文件树 + 搜索
- 协作组: Git + 团队 + 云端
- 使用 Tab 切换内容

### 4.3 新增 CommandPalette (Ctrl+K)

**功能**:
- 全局命令搜索
- 支持文件搜索、符号搜索
- 快速访问设置、技能市场、MCP工具
- 支持快捷命令 `/generate`, `/mcp`, `/run`

**触发方式**:
- `Ctrl+K` / `Cmd+K`
- ActivityBar 点击

**界面**:
```
┌──────────────────────────────────┐
│ 🔍 搜索命令、文件、符号...       │
├──────────────────────────────────┤
│ 📄 src/main.py                   │
│ 📁 pycoder/electron/src         │
│ 🧩 Skills Market                │
│ ⚙ Settings                      │
│ 🚀 Generate Project             │
│ 🧪 Run Tests                    │
└──────────────────────────────────┘
```

### 4.4 AI Panel 优化

**改进**:
- 可调整宽度（200px-600px）
- 支持折叠为图标栏
- 会话选择器更紧凑
- 输入框支持 Markdown 预览

### 4.5 底部面板整合

**整合内容**:
- Python 运行器
- Run&Fix
- 测试生成
- 调试器
- 终端
- 输出
- 问题列表

**切换方式**: 标签页切换

## 五、交互设计

### 5.1 快捷键体系

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+K` | 打开命令面板 |
| `Ctrl+B` | 切换侧边栏 |
| `Ctrl+Shift+A` | 切换AI面板 |
| `Ctrl+` `` ` `` | 切换底部面板 |
| `Ctrl+P` | 快速打开文件 |

### 5.2 拖拽调整

- 面板边缘显示拖拽手柄
- 拖拽时实时显示尺寸
- 尺寸持久化到本地存储

### 5.3 响应式行为

| 窗口宽度 | 行为 |
|----------|------|
| < 800px | AI面板自动折叠 |
| < 600px | 侧边栏自动折叠 |
| < 400px | ActivityBar 图标缩小 |

## 八、数据存储

新增 store 字段：

```typescript
interface LayoutState {
  sidebarWidth: number;      // 侧边栏宽度
  aiPanelWidth: number;      // AI面板宽度
  bottomPanelHeight: number; // 底部面板高度
  sidebarOpen: boolean;      // 侧边栏状态
  aiPanelOpen: boolean;      // AI面板状态
  bottomPanelOpen: boolean;  // 底部面板状态
}
```

持久化方式：`localStorage` 或 `electron-store`
